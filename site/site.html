CONHEÇA O PROJETO

O QUE É O "JUMP REX"?

"JUMP REX" (traduzindo do inglês significa "Dinossauro que pula") é um projeto desenvolvido pelos estudantes do Curso Técnico em Informática, CTI, da instituição Setrem. Jogo esse direcionado para todos os publicos com o intuito de estimular a coordenação motora e intelectual. O objetivo do projeto é a criação de um jogo de arduino e o desenvolvimento de um site para o próprio.
